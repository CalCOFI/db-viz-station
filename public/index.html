About CalCOFI Data Explorer

This app is better suited for viewing on a computer, mobile displays are not fully supported.

CalCOFI Data Explorer is a prototype data discovery tool developed by Nicole Xu, Aashish Krishnan, and Qiongwen (Lucy) Cao, a team of students from the University of California, Santa Barbara as part of the UCSB Data Science Capstone Program, in partnership with CalCOFI, CA SeaGrant, and Scripps Institute of Oceanography, under the direction of Dr. Erin Satterthwaite.

The application aggregates metadata from multiple marine science repositories to help users discover variables, stations, and datasets across the California Current Ecosystem.


Disclaimers

  • This app is still under development. Many features are a work in progress, such as the station variable lists which only display hydrographic data as of this moment. Please use the search bar to explore the full range of available data.
  • This application does not host the underlying scientific data. Data availability and metadata are provided by external organizations.
    • Dataset contents, coverage, and access policies may change without notice.
    • Users should consult the original data providers for authoritative information.
